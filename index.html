Memoira Assistant

Guide an interview and write up a story

This is chatbot tool to assist humans in asking questions during an interview, remembering details, and also producing writing. It is designed to ask the user questions in an interview format, with an expectation that the user will input long spoken answers to its questions. Eventually, the assistant will output a draft of a chapter on whatever topic was discussed during a session. A session ends either when the user asks for a draft, or the AI determines it has enough information to output a draft.
